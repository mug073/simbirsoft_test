Код написан с использованием фреймфорка yii.
Добавлены виджеты kartik date range picker для фильтрации по дате. А так же расширение yii2-field-range для фильтрации по диапазону в графе поиска "дом".
Фреймворк ставился на Linux систему с использованием Apche.
Так как небыла произведена тонкая настройка фреймворка не включены безусловные ссылки. Строка адреса для открытия проекта http://localhost/basic/web/index.php?r=address

По заданию реализовано:
База данных хранения адресов с задаными полями. Web-приложение выводить в табличном виде все имеющиеся адреса. Пользовательский интерфейс с использованием   Bootstrap.
Заголовки столбцов приводят к сортировке данных. При нажатии изменяется цвет, указывая на используемый в данный момент фильтр.(параметры в css) Рядом с заголовком пиктограмма указывает направление сортировки.
 Кнопка "Сброс фильтров" сбрасывет используемый фильтр путем перезагрузки страницы.
Одновременно на экран выводится по 20 записей для удобства чтения. Фильтры сохраняют свои значения вне зависимости от перехода на следующую страницу данных. Используется пагинация.

Из не реализованого:
Отсутствует локализация на английский язык.
Отсутствует пошаговое описание установки и развертывания.
